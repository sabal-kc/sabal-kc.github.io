Personal website powered by Jekyll

Template used from: https://github.com/alshedivat/al-folio

Run
---

Docker (recommended):

```bash
docker-compose up --build
```

Then open http://localhost:8080

Notes:
- The Docker setup serves on port 8080 as configured in `docker-compose.yml`.
- Use `--build` to rebuild the image if you change the `Dockerfile`.
